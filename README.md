# scc-grade
Organ sistemlerine göre SCC isimlendirme ve gradeleme için özgün eğitim/navigasyon aracı.

- Yalnızca SCC içeren veya SCC açısından pratikte özel/nadir bağlamı olan organ/protokol başlıkları listelenir.
- SCC dışı / odak dışı başlıklar listeye alınmamıştır.
- Dış PDF bağlantısı içermez; protokol bağlantıları değişirse kırık link oluşmaz.
- Bu sayfa resmi bir mesleki kuruluş ürünü veya onaylı yayını değildir.
- Nihai raporlama için güncel resmi kaynaklar ve WHO sınıflaması ayrıca kontrol edilmelidir.
